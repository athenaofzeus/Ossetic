Раскладка для шугнанского языка

Раскладка предназначена для компьютеров на ОС семейства Windows. На ОС Windows 10 раскладка работает, но некорректно отображается в панели задач. На устройствах с другими версиями Windows раскладка отображается в панели задач как HS («верхнелужицкий язык»).

При скачивании архива откроются две версии раскладки. В одной версии для набора дополнительных символов шугнанского алфавита нужно зажимать клавишу AltGr (правый Alt), в другой — клавишу Ctrl. Раскладок две, потому что пользователям могут быть удобны разные варианты; кроме того, клавиша AltGr есть не на всех устройствах.

Чтобы установить нужную раскладку, скачайте архив, распакуйте его, зайдите в распакованную папку и запустите файл setup.exe. После этого откроется установщик раскладки. По завершении установки остальные файлы можно удалить.

Раскладка устроена таким образом, чтобы ей можно было легко начать пользоваться, уже зная латинский алфавит, и без проблем переключаться между английским и шугнанским языками. За основу взята самая распространённая латинская раскладка QWERTY. Кроме символов шугнанского алфавита, в раскладку также добавлены некоторые типографские символы: среднее и длинное тире, кавычки и акут (знак ударения).
